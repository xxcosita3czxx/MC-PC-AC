
# MC-PC-AC Non-GUI

Non-gui version of mc-pc-ac


## Usage/Examples

If you use the script, youcan run this ->
```bash
python3 mc-pc-ac.py --help
```
or if you download the build for your system ->
```bash
mc-pc-ac --help
```
With Extension ofc (Does not count on Linux)
